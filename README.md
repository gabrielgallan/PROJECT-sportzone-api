# Features

## Identity

* [x] It should be able to register a new account
* [x] It should be able to authenticate with email and password
* [ ] It should be able to authenticate with GitHub
* [ ] It should be able to authenticate with Google
* [ ] It should be able to request a password recovery code
* [ ] It should be able to reset a password using a recovery code
* [x] It should be able to get the authenticated user profile
* [ ] It should be able to create an organization
* [ ] It should be able to list user organizations
* [ ] It should be able to list pending invitations
* [ ] It should be able to accept an invitation
* [ ] It should be able to decline an invitation

### As Owner

* [ ] It should be able to invite a new member
* [ ] It should be able to list organization members
* [ ] It should be able to update a member role
* [ ] It should be able to transfer organization ownership
* [ ] It should be able to remove a member from the organization

---

## Notifications

* [ ] It should be able to list user notifications
* [ ] It should be able to get notification details
* [ ] It should be able to mark a notification as read
* [ ] It should be able to mark all notifications as read
* [ ] It should be able to count unread notifications

---

## Reviews

* [ ] It should be able to create a court review
* [ ] It should be able to update an own review
* [ ] It should be able to delete an own review
* [ ] It should be able to list court reviews
* [ ] It should be able to get a review by id
